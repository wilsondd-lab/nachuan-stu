# 纳川图解 · 内容创作指南

> AI 创作信息图内容时的指导原则和方法论。好的内容是一张图能否帮到学生的关键。

---

## 一、内容创作原则

### 1.1 准高中生定位

面向**初三升高一**的学生，内容难度要：
- **不超纲** — 以初中课标为核心，适当衔接高中
- **有深度** — 不只是重复课本，要帮学生"开窍"
- **重理解** — 讲清为什么，不只是背结论
- **有体系** — 帮学生建立知识框架，而不是零散知识点

### 1.2 一张图一个核心

- 每张信息图只讲**一个核心知识点**
- 不要试图在一张图里塞太多内容
- 所有内容围绕核心展开，不跑题
- 学生看完后能用一句话说清这张图讲了什么

### 1.3 记忆导向

信息图的目的是**帮学生记住**。创作时要思考：
- 怎么讲学生更容易理解？
- 怎么组织学生更容易记住？
- 用什么比喻学生更有共鸣？
- 一句话总结能不能脱口而出？

## 二、知识点拆解方法

### 2.1 五问拆解法

拿到一个知识点，问自己五个问题：

| 问题 | 说明 | 对应区块 |
|------|------|----------|
| **是什么？** | 定义、概念、本质 | 概念卡片 |
| **怎么算？** | 公式、定理、计算方法 | 公式区 |
| **怎么用？** | 适用条件、常见题型 | 应用举例 |
| **容易错什么？** | 常见误区、易混点 | 易错点区 |
| **和什么有关系？** | 前后知识联系 | 总结区 |

### 2.2 概念卡片撰写

每张概念卡片要包含：
- **名称** — 概念的标准名称（+符号/字母）
- **一句话定义** — 用最简洁的话讲清楚是什么
- **关键词加粗** — 核心词用粗体突出

**好的例子：**
> 速度 v
> 描述物体运动**快慢**的物理量。单位时间内通过的位移。

**不好的例子：**
> 速度是一个很重要的物理量，在物理学中我们经常用到，它可以告诉我们物体运动的情况...

### 2.3 易错点撰写

易错点要**具体、真实、有价值**：
- 是学生真的会犯的错误，不是凭空想象的
- 指出错误是什么，更要说清为什么错
- 给出正确的理解方式
- 最好能总结成一句话提醒

**格式模板：**
```
易错点X：[错误的说法/理解]
错！[为什么错 / 反例]
正确理解：[一句话说清正确的认识]
```

**好的例子：**
> 易错点1：速度为零 = 加速度为零？
> 错！竖直上抛到最高点时，速度为零，但加速度仍是 g = 9.8 m/s²。
> 加速度只和速度的"变化量"有关，和速度本身大小无关。

## 三、类比和比喻设计

### 3.1 为什么要用类比

- 抽象概念 → 具体事物
- 陌生知识 → 熟悉经验
- 降低理解门槛
- 增强记忆效果

### 3.2 好类比的标准

- **熟悉** — 学生日常生活中接触过
- **贴切** — 核心特征对应准确，不误导
- **简洁** — 一句话能说明白
- **有趣** — 有画面感，容易记住

### 3.3 常见类比方向

| 抽象概念 | 类比方向 | 示例 |
|----------|----------|------|
| 速度 vs 加速度 | 开车 | 速度是车速，加速度是踩油门 |
| 电压 | 水压 | 电压像水压，推动电荷流动 |
| 电流 | 水流 | 电流像水流，电荷像水滴 |
| 电阻 | 管道阻力 | 电阻像水管粗细，粗的阻力小 |
| 分子热运动 | 人群 | 温度越高，人跑得越快 |
| 原子结构 | 太阳系 | 原子核像太阳，电子像行星 |
| 基因 | 蓝图 | 基因像建筑蓝图，指导蛋白质建造 |
| 酶 | 催化剂/钥匙 | 酶像钥匙，打开反应大门 |
| 密度 | 拥挤程度 | 密度大 = 人挤人，密度小 = 很宽松 |
| 压强 | 压力分布 | 同样的力，尖的地方压强大 |

### 3.4 使用类比的注意事项

- **类比不是等同** — 要说明类比的边界，避免误导
- **只用核心特征类比** — 不追求面面俱到
- **一张图1-2个类比就够** — 太多反而混乱
- **放在概念引入处** — 帮助学生快速建立直觉

## 四、标题撰写

### 4.1 主标题

- 直接点题，不绕弯子
- 10个字以内最佳
- 核心概念加粗或高亮
- 可用 "vs" "·" 等符号连接

**好的标题：**
- 速度 vs 加速度
- 欧姆定律
- 光合作用全解
- 勾股定理

**不好的标题：**
- 关于速度和加速度的一些知识
- 物理学习心得分享
- 第三章知识点总结

### 4.2 副标题

- 补充说明，增加价值感
- 15-25个字
- 可以是：定位、痛点、效果、金句

**好的副标题：**
- 运动学基础 · 两个最容易搞混的物理量
- 一张图搞懂 · 电学的核心定律
- 再也不怕记混了 · 3分钟彻底理解

## 五、总结金句

### 5.1 为什么要有总结金句

- 帮学生用一句话记住核心
- 可以脱口而出，方便复习
- 信息图的"记忆锚点"

### 5.2 好金句的标准

- **短** — 1-2句话，不超过50字
- **准** — 准确传达核心意思
- **顺** — 朗朗上口，有节奏感
- **记得住** — 看过一遍就能复述

### 5.3 金句模板

**对比型：**
> A是XX，B是YY；A和B的区别是ZZ。

**口诀型：**
> XX变，YY不变；ZZ条件要记全。

**场景型：**
> 当你遇到XX问题，就想YY方法。

**公式型：**
> 核心公式 + 一句话解释

### 5.4 示例

**速度与加速度：**
> 速度说的是跑多快，加速度说的是速度变得多快；
> 二者方向相同就加速，方向相反就减速。

**欧姆定律：**
> 电流等于电压除以电阻，
> 电压越大电流越大，电阻越大电流越小。

**物态变化：**
> 固态变液叫熔化，液态变气叫汽化，
> 固态直接变气态，名字叫做升华。
> 三个过程都吸热，反过来就都放热。

## 六、视觉内容规划

### 6.1 什么时候需要图示

不是所有知识点都需要复杂图示。判断标准：
- **空间关系** — 位置、结构、路径 → 需要图示
- **变化过程** — 运动、反应、循环 → 需要图示
- **抽象概念** — 需要可视化帮助理解 → 需要图示
- **纯文字定义** — 不需要图示

### 6.2 图示设计原则

- **简单** — 只画必要的元素，不画无关装饰
- **标注清楚** — 每个重要部分都有标注
- **颜色有意义** — 不同颜色代表不同含义，保持一致
- **方向正确** — 物理方向、化学流向不能画反
- **比例合理** — 各部分大小关系符合实际

### 6.3 常见图示类型

| 类型 | 适用场景 | 示例 |
|------|----------|------|
| 示意图 | 物理过程、运动轨迹 | 小车受力图、运动示意 |
| 坐标图 | 函数关系、变化规律 | v-t图像、s-t图像 |
| 结构图 | 物体组成、器官结构 | 细胞结构、原子结构 |
| 装置图 | 实验装置、仪器 | 加热装置、电路连接 |
| 流程图 | 步骤、过程、循环 | 物态变化、消化过程 |
| 对比图 | 两个概念对比 | 速度vs加速度示意图 |

## 七、学科内容特点

### 7.1 物理

- **重公式** — 每个核心知识点都要有公式
- **重图示** — 受力图、运动图、电路图是核心
- **重单位** — 单位和公式一样重要
- **重条件** — 每个公式都有适用条件
- **重方向** — 矢量的方向是易错点

### 7.2 化学

- **重符号** — 元素符号、化学式、化学方程式
- **重实验** — 实验装置、操作步骤、现象
- **重规律** — 周期律、金属活动性、反应规律
- **重微观** — 分子、原子、离子层面的解释

### 7.3 生物

- **重结构** — 细胞、组织、器官、系统
- **重过程** — 光合、呼吸、消化、循环
- **重功能** — 每个结构对应什么功能
- **重联系** — 各系统之间如何配合

### 7.4 数学

- **重公式** — 定理、公式、法则
- **重图形** — 几何图形、函数图像
- **重推导** — 公式怎么来的
- **重应用** — 典型题型和解题方法

### 7.5 语文

- **重积累** — 字词、古诗文、文学常识
- **重方法** — 阅读方法、写作技巧
- **重体系** — 知识框架、文体分类
- **重辨析** — 易混概念、易错点

## 八、内容量控制

### 8.1 一张图的内容上限

| 区块 | 内容量 |
|------|--------|
| 标题 | 主标题10字 + 副标题20字 |
| 概念卡片 | 3张，每张约30-50字 |
| 公式区 | 2-3个核心公式 |
| 易错点 | 2-3个，每个约40-60字 |
| 总结 | 1-2句话，30-50字 |
| **总字数** | **约300-500字** |

### 8.2 为什么控制内容量

- 竖版信息图空间有限
- 太多文字 = 看了就忘
- 少即是多 — 3个核心点 > 10个零散点
- 学生的注意力是稀缺资源

### 8.3 内容取舍原则

- 保留：核心概念、关键公式、高频易错点
- 舍弃：细枝末节、扩展知识、太深入的推导
- 不确定时：问自己"这个对理解核心概念必要吗？"

## 九、语言风格

### 9.1 目标

像一个**会教书的学长/学姐**，而不是冷冰冰的课本。

### 9.2 具体要求

- **简洁** — 一句话说清的不说两句
- **准确** — 科学术语不能含糊
- **亲切** — 用"你"而不是"学生"
- **有温度** — 理解学生的困难，不说"这都不会"
- **有力量** — 鼓励学生，给信心

### 9.3 避免的说法

- "很简单"、"很容易" — 学生会觉得自己笨
- "必须"、"一定要" — 太生硬
- 长篇大论的定义 — 没人愿意读
- 抽象的理论堆砌 — 没有具体例子
