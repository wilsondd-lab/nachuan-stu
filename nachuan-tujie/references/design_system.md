# 纳川图解 · 设计系统

> 所有信息图共享的视觉规范。AI 创作时必须严格遵循，保证视觉一致性和专业质感。

---

## 一、画布规格

| 属性 | 值 | 说明 |
|------|-----|------|
| 尺寸 | 1080 × 1920 px | 9:16 竖版，适配手机全屏 |
| 内边距 | 左右 56px，上下 60px | 内容安全区 |
| 背景色 | #F8F9FA（极浅灰） | 全局底色，也可使用主题色浅色调渐变 |
| 分辨率 | 72dpi（屏幕显示） | PNG 输出默认 |

**注意：** 所有元素必须在 1080×1920 画布内，不能溢出。内容区域约为 968×1800。

## 五、字体规范

### 字体栈

```css
/* 中文字体栈 */
font-family: 'PingFang SC', 'Microsoft YaHei', 'Noto Sans CJK SC', sans-serif;

/* 等宽字体栈（公式/代码） */
font-family: 'SF Mono', 'Menlo', 'Consolas', monospace;
```

**说明：**
- macOS 默认有 PingFang SC（苹方）
- Windows 默认有 Microsoft YaHei（微软雅黑）

## 六、间距系统（8px 基准）

| 变量名 | 像素值 | 用途 |

| `--space-xs` | 8px | 最小间距，图标与文字间隙 |

| `--space-lg` | 32px | 大间距，模块之间间距 |
| `--space-xl` | 48px | 超大间距，主要区块之间 |

## 十、配色使用原则

1. **主色统一** — 一张图只用一套主色系，保持视觉统一
2. **功能色辅助** — 警告、成功、强调色作为功能色使用，不超过画面的 15%
3. **浅色背景** — 大面积使用浅色（主色-bg），深色只用于文字和小面积装饰
4. **对比度达标** — 文字与背景对比度不低于 4.5:1，确保可读性
5. **渐变克制** — 渐变只用于标题栏、总结卡等重点区域，不超过 2 处
6. **中性色为主** — 60% 以上面积使用中性色（白、灰、浅背景）
