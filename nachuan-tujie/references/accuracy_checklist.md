# 纳川图解 · 准确性检查清单

> 每一张信息图生成后，AI 必须按照本清单逐项自检。不通过的项目必须修改，直到全部通过才能渲染交付。

---

## 一、通用检查（所有学科必查）

### 1.1 文字准确性

- [ ] **无错别字** — 通读全文，检查每个词语
- [ ] **术语正确** — 学科术语使用教材标准表述
- [ ] **单位正确** — 物理量单位使用国际标准单位，符号大小写正确
  - 正确：m（米）、s（秒）、kg（千克）、N（牛顿）、Pa（帕斯卡）
  - 注意：人名命名的单位首字母大写（N, A, V, Ω），普通单位小写（m, s, g）
- [ ] **符号规范** — 数学符号、希腊字母使用正确
- [ ] **公式完整** — 公式不缺项、不错位
- [ ] **数字准确** — 所有数字（常数、系数、指数）正确
  - g = 9.8 m/s²（或题目指定 10 N/kg）
  - π ≈ 3.14
  - c = 3×10⁸ m/s（真空中光速）
  - 水的密度 = 1.0×10³ kg/m³

### 1.2 图形准确性

- [ ] **方向正确** — 箭头方向与物理/数学意义一致
  - 重力方向：竖直向下（不是"垂直向下"）
  - 支持力方向：垂直于接触面向上
  - 电流方向：正电荷移动方向（从正极到负极）
- [ ] **比例准确** — 几何图形、函数图像的比例关系正确
- [ ] **标注清晰** — 图形各部分有明确标注，没有歧义
- [ ] **SVG 无溢出** — 所有图形元素在 viewBox 内

### 1.3 知识准确性

- [ ] **定义准确** — 概念定义与教材表述一致
- [ ] **条件完整** — 公式/定理的适用条件没有遗漏
- [ ] **因果正确** — 因果关系不颠倒
- [ ] **举例恰当** — 生活实例符合科学原理
- [ ] **易错点准确** — 指出的常见错误确实是学生常犯的

### 1.4 排版与显示

- [ ] **文字不溢出** — 所有文字在卡片/容器内
- [ ] **元素不重叠** — 各区块之间没有重叠
- [ ] **在画布内** — 所有内容在 1080×1920 范围内
- [ ] **字体可读** — 最小字号不低于 14px
- [ ] **对比度达标** — 文字与背景颜色对比度足够

## 二、物理学科检查清单

### 2.1 力学

- [ ] 速度公式 v = s/t，单位 m/s，正确
- [ ] 加速度公式 a = Δv/t，单位 m/s²，正确
- [ ] 重力公式 G = mg，g = 9.8 N/kg（或题目指定），正确
- [ ] 密度公式 ρ = m/V，单位 kg/m³ 或 g/cm³，正确
- [ ] 压强公式 p = F/S，单位 Pa，正确
- [ ] 液体压强 p = ρgh，h 是深度（从液面往下算），正确
- [ ] 浮力公式 F浮 = ρ液gV排，正确
- [ ] 功的公式 W = Fs，单位 J，正确
- [ ] 功率公式 P = W/t = Fv，单位 W，正确
- [ ] 杠杆平衡条件 F₁L₁ = F₂L₂，正确
- [ ] 牛顿第一定律：惯性定律，不是"力是维持运动的原因"
- [ ] 平衡力 vs 相互作用力的区别清晰
- [ ] 摩擦力方向：与相对运动（趋势）方向相反

### 2.2 运动学图示

- [ ] s-t 图像：斜率 = 速度，水平直线表示静止
- [ ] v-t 图像：斜率 = 加速度，水平直线表示匀速
- [ ] 速度方向和加速度方向标注正确
- [ ] 位移方向和路径区分清楚

### 2.3 电学

- [ ] 电流公式 I = U/R（欧姆定律），正确
- [ ] 电功公式 W = UIt = I²Rt = U²t/R，正确
- [ ] 电功率公式 P = UI = I²R = U²/R，正确
- [ ] 焦耳定律 Q = I²Rt，正确
- [ ] 串联电路：电流处处相等，总电压等于各部分电压之和
- [ ] 并联电路：各支路电压相等，干路电流等于各支路电流之和
- [ ] 电路图符号规范（电池正负极、开关、灯泡、电阻、电流表、电压表）
- [ ] 电流表串联、电压表并联，不能混淆
- [ ] 电流方向：外电路从正极流向负极

### 2.4 光学

- [ ] 光的直线传播条件：同种均匀介质
- [ ] 反射定律：三线共面、两线分居、两角相等
- [ ] 折射规律：从空气斜射入水/玻璃，折射角小于入射角
- [ ] 凸透镜成像规律表格正确（u>2f、u=2f、f<u<2f、u=f、u<f）
- [ ] 光路图中箭头方向正确（光线从物体出发）

### 2.5 热学

- [ ] 温度单位：摄氏度（℃），热力学温度 K
- [ ] 热量公式 Q = cmΔt，正确
- [ ] 热值公式 Q = mq（固体液体）或 Q = qV（气体），正确
- [ ] 物态变化名称和吸放热对应正确
  - 熔化：固→液，吸热
  - 凝固：液→固，放热
  - 汽化：液→气，吸热
  - 液化：气→液，放热
  - 升华：固→气，吸热
  - 凝华：气→固，放热

## 三、数学学科检查清单

### 3.1 代数

- [ ] 公式书写正确，没有符号错误
- [ ] 正负号正确
- [ ] 分数的分子分母位置正确
- [ ] 方程的解代入验证正确
- [ ] 函数表达式与图像对应正确

### 3.2 几何

- [ ] 几何图形的边长、角度标注正确
- [ ] 直角标记（┐）使用正确
- [ ] 全等/相似符号正确：≌ / ∽
- [ ] 三角形内角和 = 180°
- [ ] 勾股定理：a² + b² = c²（直角三角形）
- [ ] 圆的周长 C = 2πr，面积 S = πr²
- [ ] 圆柱体积 V = πr²h，圆锥体积 V = (1/3)πr²h

### 3.3 函数图像

- [ ] 一次函数 y = kx + b，直线走向与 k 的正负一致
  - k > 0：从左下到右上
  - k < 0：从左上到右下
- [ ] 反比例函数 y = k/x，双曲线位置与 k 的正负一致
- [ ] 二次函数 y = ax² + bx + c，开口方向与 a 的正负一致
  - a > 0：开口向上
  - a < 0：开口向下
- [ ] 顶点坐标、对称轴位置正确
- [ ] 坐标轴标注正确（横轴x、纵轴y，原点O）

## 四、化学学科检查清单

### 4.1 基本概念

- [ ] 元素符号大小写正确（首字母大写，第二个小写）
  - 正确：H, He, O, Al, Fe, Cu
  - 错误：he, al, FE
- [ ] 化学式书写正确（化合价代数和为零）
  - 水 H₂O、二氧化碳 CO₂、氯化钠 NaCl
- [ ] 化学方程式配平正确
- [ ] 反应条件标注正确（加热△、点燃、催化剂、高温）
- [ ] 气体↑和沉淀↓符号使用正确

### 4.2 原子结构

- [ ] 原子序数 = 质子数 = 核电荷数 = 核外电子数（原子中）
- [ ] 相对原子质量 ≈ 质子数 + 中子数
- [ ] 电子层排布正确（2n²规则，最外层不超过8个）
- [ ] 金属元素最外层电子数 < 4，非金属 ≥ 4，稀有气体 = 8（He=2）

### 4.3 实验装置

- [ ] 试管加热时管口略向下倾斜（防止冷凝水倒流）
- [ ] 长颈漏斗下端伸入液面以下（液封）
- [ ] 导管伸入集气瓶的位置正确（排气法 vs 排水法）
- [ ] 酒精灯用外焰加热
- [ ] 实验装置图中各仪器比例合理

## 五、生物学科检查清单

### 5.1 细胞

- [ ] 动物细胞：细胞膜、细胞质、细胞核、线粒体
- [ ] 植物细胞：细胞壁、细胞膜、细胞质、细胞核、液泡、叶绿体、线粒体
- [ ] 细胞各结构功能描述正确
- [ ] 细胞分裂、细胞分化的概念不混淆

### 5.2 生理过程

- [ ] 光合作用公式：二氧化碳 + 水 → 有机物 + 氧气（条件：光、叶绿体）
- [ ] 呼吸作用公式：有机物 + 氧 → 二氧化碳 + 水 + 能量
- [ ] 光合作用场所：叶绿体；呼吸作用场所：线粒体
- [ ] 蒸腾作用：水分以气体状态从植物体内散发出去
- [ ] 消化系统、呼吸系统、循环系统的结构和功能对应正确

### 5.3 遗传与进化

- [ ] DNA 是主要的遗传物质
- [ ] 基因是有遗传效应的 DNA 片段
- [ ] 染色体 = DNA + 蛋白质
- [ ] 显性基因 / 隐性基因的概念正确
- [ ] 自然选择学说的核心要点正确

## 八、自检流程

### 第一步：通读全文
从头到尾读一遍所有文字，检查错别字和语句通顺。

### 第二步：公式核查
逐个公式检查：符号、上下标、单位、适用条件。

### 第三步：图形审查
检查所有SVG图示：方向、比例、标注、颜色。

### 第四步：知识核对
对照本清单中学科特定的检查项，逐一确认。

### 第五步：排版验证
确认所有内容在 1080×1920 画布内，没有溢出和重叠。

## 九、不合格的处理

| 问题类型 | 处理方式 |
|----------|----------|
| 错别字/文字错误 | 直接修改文字 |
| 公式错误 | 修正公式，重新检查推导 |
| 图形方向错误 | 修改 SVG 坐标，重新绘制 |
| 比例不对 | 调整 SVG 尺寸和坐标 |
| 知识错误 | 重新梳理知识点，修正内容 |
| 排版溢出 | 调整字号、间距、布局 |
| 颜色对比度不够 | 调整颜色，确保可读性 |

**原则：** 准确性优先于美观。如果准确性和美观冲突，先保证准确，再考虑美观。
