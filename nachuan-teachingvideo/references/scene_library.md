# 场景库使用指南

## 概述

`templates/` 目录下有6个开箱即用的场景模板，每个都是一个独立的HTML片段，包含结构、样式插槽和GSAP入场动画代码。

## 6个场景模板

### 1. scene-title.html — 标题引入场景

**用途**：视频第一段，用来引入主题，制造悬念。

**包含元素**：
- 主题图标（SVG占位）
- 大标题
- 子概念胶囊横排
- 副标题

**动画**：
- 标题从下方浮入（power3.out）
- 图标缩放弹入（back.out）
- 胶囊逐个错峰出现（stagger）
- 副标题淡入
- 环境动效：图标呼吸

**使用场景**：
- s1 标题引入段
- 替换图标 SVG、标题文字、胶囊文字

---

### 2. scene-compare.html — 左右对比场景

**用途**：两个相似概念的对比，帮助学生厘清区别。

**包含元素**：
- 左侧概念区（标题 + 示意图 + 要点卡片）
- 中间 VS 徽章
- 右侧概念区（标题 + 示意图 + 要点卡片）

**动画**：
- 左侧从左滑入
- 右侧从右滑入
- VS 徽章居中弹出
- 要点逐条出现
- 环境动效：VS 徽章呼吸

**典型对比**：
- 速度 vs 加速度
- 音调 vs 响度
- 串联 vs 并联
- 有丝分裂 vs 减数分裂

---

### 3. scene-formula.html — 公式推导场景

**用途**：核心公式讲解，变量含义解释，推导过程展示。

**包含元素**：
- 核心公式（大号显示）
- 推导步骤（可选，3步）
- 变量含义卡片
- 要点卡片

**动画**：
- 公式缩放弹入
- 推导步骤逐条滑入
- 变量卡片从右滑入
- 要点卡片从下滑入
- 环境动效：公式脉冲

**变量着色原则**：
- 每个变量用不同颜色（主色、次强调、中性色）
- 公式中的变量和解释卡片中的变量颜色对应
- 方便学生建立视觉关联

---

### 4. scene-example.html — 生活例子场景

**用途**：用学生熟悉的生活场景解释抽象概念。

**包含元素**：
- 生活场景插图（SVG）
- 问题气泡
- 知识点联系卡片
- 拓展例子卡片

**动画**：
- 场景整体淡入
- 关键元素逐个驶入/出现
- 问题气泡弹出
- 解释卡片滑入
- 标注浮现
- 环境动效：场景元素轻微浮动

**生活例子库**：

| 知识点 | 生活例子 |
|--------|---------|
| 速度 | 汽车时速表、跑步、高铁 |
| 加速度 | 汽车起步、电梯启动、过山车 |
| 声现象 | 敲鼓、吉他弦、回音壁 |
| 光现象 | 彩虹、镜子、放大镜 |
| 电路 | 手电筒、家庭电路、红绿灯 |
| 光合作用 | 树叶、种菜、森林 |
| 密度 | 游泳浮起来、热气球、油浮水面 |

---

### 5. scene-quiz.html — 问答揭晓场景

**用途**：测试学生理解，制造悬念后揭晓答案。

**包含元素**：
- 问题
- 4个选项（2x2网格）
- 正确答案高亮
- 答案解释

**动画**：
- 问题从上方浮入
- 选项逐个弹出
- 停顿后正确答案高亮（边框+发光+缩放）
- 错误选项淡化
- 答案解释从下滑入
- 环境动效：正确答案脉动

**设计原则**：
- 问题要简单，考察核心概念
- 干扰项要是常见错误
- 答案解释要简洁，点出关键
- 可以用在 s6 段，作为知识点应用

---

### 6. scene-summary.html — 总结收尾场景

**用途**：视频最后一段，回顾要点，强化记忆。

**包含元素**：
- 核心公式/大数字
- 注释文字
- 要点回顾卡片
- 概念徽章横排

**动画**：
- 大数字缩放弹入
- 注释淡入
- 卡片从下滑入
- 徽章逐个弹入
- 环境动效：大数字呼吸
- 收尾淡出（仅video模式）

**徽章设计**：
- 4-5个核心概念徽章
- 每个徽章有图标和名称
- 图标用简洁符号（→ v s t ∿ 等）
- 颜色统一用主色

## 模板使用方法

### 方法一：直接复制

1. 打开 `templates/scene-xxx.html`
2. 复制 `<div class="scene ...">...</div>` 部分
3. 粘贴到 index.html 的对应场景位置
4. 替换内容（文字、SVG）
5. 复制底部的 GSAP 动画函数体到 sceneN(t)

### 方法二：用 scaffold 生成骨架后替换

1. 运行 `scaffold_video.py` 生成骨架
2. 从模板复制需要的场景结构
3. 替换掉骨架中的对应场景

### 方法三：混合使用

不同场景用不同模板，自由组合：
- s1: scene-title
- s2: scene-formula
- s3: scene-compare
- s4: scene-example
- s5: scene-formula
- s6: scene-quiz
- s7: scene-summary

## 自定义配色

所有模板使用 CSS 变量控制配色，修改 `:root` 里的9个变量即可一键换主题：

```css
:root {
  --primary:   #D63B26;   /* 主色 */
  --accent:    #F08A24;   /* 次强调 */
  --highlight: #FFD166;   /* 浅辅 */
  --ink:       #26303A;   /* 正文色 */
  --neutral:   #64748B;   /* 中性色 */
  --soft:      #FFFDF8;   /* 卡片底 */
  --line:      #A3AEBB;   /* 结构线 */
  --bg:        #FBF5EB;   /* 背景 */
  --series:    #64748B;   /* 章节标 */
}
```

### 7套预设配色

| 主题 | 主色 | 适用学科 |
|------|------|---------|
| sound | 暖橙 `#E8631C` | 声学、波动 |

| mechanics | 暖红 `#D63B26` | 力学、运动 |
| electricity | 深蓝 `#0D3B66` | 电学、电路 |

## 动画设计原则

### 入场节奏

每个场景的入场顺序遵循"讲解顺序"：
1. 背景/框架先出现
2. 主体元素（示意图）逐步绘制
3. 标注/重点弹出
4. 文字卡片滑入
5. 收尾停顿 + 环境动效

### 动效四原则

1. **缓入缓出**：入场用 ease out，移动用 ease in out
2. **预备动作**：关键元素入场前微微反向再弹出（back.out）
3. **超出回弹**：标题/数字用 back.out(1.5)，卡片用 power3.out
4. **分层错峰**：多元素 stagger 0.09-0.25s

### 环境动效

每个场景选一种环境动效即可，不要太多：
- 呼吸（scale yoyo）—— 适合图标、徽章
- 浮动（y yoyo）—— 适合场景元素
- 脉冲（scale + opacity）—— 适合重点强调
- 漂移（x repeat）—— 适合波形、流动

### 硬规则

- 一律用 `tl.fromTo()`，不用 `tl.from()`
- 环境动效必须挂在 `tl` 上
- 不用 `Math.random()`，保持渲染确定性
- 循环动效用有限 repeat，不用 `repeat: -1`
