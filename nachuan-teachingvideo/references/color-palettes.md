# 调色板指南

> 核心原则：**配色服务主题**。不同物理/学科主题用不同色系，不要全系列一个调。
> 通常 **主色 = 主题核心元素**，**辅色 = 结构/对比**。
> 背景统一用浅暖色（`#FFFCF0` 或 `#FFF6E5`），文字统一用深色（深棕或深蓝）。

## 已验证的调色板

### 1. 声现象（cream + 橙）— 已验证

适用：声学、波动、音乐、振动、声音三要素

```python
PRIMARY   = "#E8631C"   # 主橙 (10% 强调)
ACCENT    = "#F4A24C"   # 亮橙 (对比元素)
HIGHLIGHT = "#FFD166"   # 黄
INK       = "#22303C"   # 蓝黑正文 (冷, 与暖主色对比)
NEUTRAL   = "#5E7183"   # 冷灰青 (kicker/标签)
SOFT      = "#FFFDF6"   # 卡片底 (近白)
LINE      = "#9DAEBB"   # 结构线 (冷)
BG        = "#FFF6E5"   # cream
SERIES    = "#5E7183"   # 章节标 (冷, 与 NEUTRAL 同值保证对比度)
```

### 7. 数学 / 几何（莫兰迪雾紫）— 已验证（勾股定理）

适用：几何变换、函数、统计

数学主题走**低饱和莫兰迪变体**：米白暖底 + 雾紫主色 + 灰灰蓝中性 + 焦糖暖点缀。

```python
PRIMARY   = "#7A6296"   # 雾紫加深 (公式/大数字/编号)
ACCENT    = "#B98A5C"   # 焦糖 (低饱和暖, 对比元素, 保住冷暖对比)
HIGHLIGHT = "#D8D1E6"   # 浅薰衣
INK       = "#433D50"   # 深灰紫正文
NEUTRAL   = "#6B7889"   # 灰灰蓝加深 (kicker/标签)
SOFT      = "#FCFAF6"   # 卡片底
LINE      = "#B9C2CE"   # 结构线 (浅灰蓝)
BG        = "#F7F3ED"   # 米白
SERIES    = "#6B7889"   # 章节标
```

**低饱和的适用边界**：莫兰迪化只给数学/人文向主题（数学/语文/历史），气质是"纸上推演"；
物理类主题（声/光/电/热）保持高饱和强调色不动 —— 波形、电流箭头、能量高亮需要
"能量感"，低饱和会糊，视频压缩后更糊。低饱和永远只进 60% 背景层和 30% 结构层，
物理主题的 10% 强调层不碰。

## CSS 变量映射（index.html 的 :root）

配音视频和无声动图**共用同一套色值**（同一个 index.html）。做 **60-30-10 分层**：
60% 背景米色 + 30% 冷中性结构 + 10% 主色强调。

关键：`--ink / --neutral / --line / --series` 用**与主色互补的冷中性色**，不要全画面一个色系
——主色只出现在示意图主体、公式、大数字、编号上，越少越强。

| CSS 变量 | 配色角色 | 用途 |
|---|---|---|
| `--primary` | 主色 | 10% 强调：示意图主体、公式、大数字、chip 编号、卡片左边线 |
| `--accent` | 次强调 | 对比波形/元素 |
| `--highlight` | 浅辅 | 高亮/浅色块 |
| `--ink` | 冷化深色 | 正文、字幕：偏蓝黑，与暖主色形成温差 |
| `--neutral` | 冷中性 | kicker、标签、副标题、卡片边线 |
| `--line` | 冷中性 | 网格纹理、结构线 |
| `--bg` | 背景 | 场景背景 (60%) |
| `--series` | 冷中性 | 章节角标、ghost 底纹编号 |
| `--soft` | 近白 | 卡片底色 |

例：声现象（暖橙主色 → 配冷灰青中性）：

```
--primary:#E8631C; --accent:#F4A24C; --ink:#22303C; --neutral:#5E7183;
--line:#9DAEBB; --series:#5E7183; --bg:#FFF6E5; --soft:#FFFDF6;
```

冷中性怎么选：暖主色（橙/红/黄）→ 蓝灰系（#5E7183 一族）；冷主色（蓝/紫）→ 暖灰系
（#8A7A6B 一族）；绿主色 → 中灰偏棕。原则是中性色的色温与主色**相反**。

## 复用规则

- **`SERIES` 色**（右上角章节标识）：选 **浅色** 版本，不抢主色戏
- **`LINE` 色**（结构线、镜面、法线、虚线）：选 **中明度** 版本，区分于主色
- **真实物理元素颜色优先**：红色激光保持红、蓝色水保持蓝、绿色植物保持绿
- **避免**：主色 = 辅色（无对比）/ 文字与背景同明度（看不清）/ 3 色以上高饱和（杂乱）
