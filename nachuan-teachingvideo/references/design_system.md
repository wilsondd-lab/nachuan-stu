# 设计系统 · Design System

> 纳川教学视频的视觉规范：配色、字体、动效、间距。
> 所有视频统一遵循，确保品牌一致。

---

## 一、配色系统

### 核心原则：60-30-10 + 莫兰迪低饱和

- **60% 背景色**：暖米白 / 浅灰（大面积底色，柔和不刺眼）
- **30% 中性色**：正文文字、结构线（清晰可读）
- **10% 强调色**：主色 + 次强调（用于标题、重点、图标）

莫兰迪色系特点：低饱和度、加灰调、柔和高级、长时间看不累。

## 二、字体系统

### 字体栈

```css
font-family: "Heiti SC", "PingFang SC", "Noto Sans CJK SC", sans-serif;
```

- **首选**：Heiti SC（ macOS 系统黑体，清晰有力）
- **备选**：PingFang SC（苹果系统字体，现代简洁）
- **兜底**：Noto Sans CJK SC（跨平台开源黑体）

### 装饰字体（标题、公式、大数字）

```css
font-family: "Songti SC", serif;
```

- 宋体用于标题和公式，增加学术感和分量感
- 大数字（英雄数字）一律用宋体 900 粗体

### 字号层级

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| `t-hero` | 140px | 900 | 场景大标题、英雄数字 |
| `t-title` | 54px | 800 | 场景内标题、卡片大标题 |
| `t-body` | 30-32px | 正常 | 正文、卡片文字 |
| `t-label` | 24-28px | 600-700 | 标签、kicker、章节角标 |
| 公式 | 36-48px | 800-900 | 公式行（宋体） |
| 字幕 | 32-34px | 600 | 底部字幕 |

### 行高
- 正文：1.55
- 标题：1.1
- 卡片内文字：1.5

## 三、间距系统

### 8pt 网格

所有间距都是 8 的倍数：
- 4px（微调）
- 8px / 12px / 16px / 24px / 32px / 48px / 64px / 80px

### 场景内边距

```css
.scene-inner {
  padding: 140px 120px 180px;
}
```

- 顶部 140px（留出 section-chip 和 chapter-label 空间）
- 左右 120px（安全边距，避免内容贴边）
- 底部 180px（留出字幕空间 + 呼吸感）

### 卡片间距
- 卡片内边距：24-32px
- 卡片之间 gap：28-32px
- 卡片圆角：18-20px

## 四、动效规范

### 动效四原则

| 原则 | 说明 | 实现方式 |
|------|------|---------|
| **缓入缓出** | 运动不是匀速的，有加速和减速 | `ease: "power2.out"` / `"power3.out"` |
| **预备动作** | 入场前先微微反向动一下，再弹出 | `ease: "back.out(1.5)"` |
| **超出回弹** | 关键元素到达终点后轻微超出再弹回 | `ease: "back.out(1.5~2)"` |
| **分层错峰** | 多元素不是同时出现，而是按逻辑顺序依次入场 | `stagger: 0.09~0.25` |

### 常用缓动函数

| 缓动 | 用途 | 时长参考 |
|------|------|---------|
| `power3.out` | 卡片滑入、大元素入场 | 0.6-1.0s |
| `power2.out` | 次要元素、文字入场 | 0.4-0.6s |
| `back.out(1.5)` | 按钮、徽章、小元素弹出 | 0.4-0.6s |
| `back.out(2)` | 强调元素、数字弹跳 | 0.5-0.7s |
| `sine.out` | 淡入淡出、透明度变化 | 0.3-0.7s |
| `power2.inOut` | SVG 描边绘制、转场 | 0.5-0.8s |

### 动效时长规范

| 元素类型 | 入场时长 | 出场时长 |
|----------|---------|---------|
| 大标题 / 大数字 | 0.8-1.2s | 0.4-0.6s |
| 信息卡片 | 0.6-0.8s | 0.3-0.4s |
| 小元素 / 徽章 | 0.4-0.6s | 0.2-0.3s |
| SVG 描边绘制 | 0.6-1.5s | - |
| 转场（模糊叠化） | 0.5s | - |
| 字幕淡入 | 0.35s | 0.25s |

### 环境动效（停留帧）

动画结束后不要立刻转场，给画面停留时间：
- 关键信息停留 1.5-2.5 秒
- 停留期间加一种细微环境动效（选一种即可）：
  - **呼吸**：元素轻微缩放（1 → 1.03 → 1，2s 周期）
  - **漂移**：元素上下轻微浮动（y: ±5px，3s 周期）
  - **波形流动**：波形持续平移（线性 repeat）
  - **光晕脉动**：发光强度周期性变化

**规则**：一场景一种环境动效就够了，不要每个元素都在动。

## 五、组件规范

### 信息卡片 (info-card)

```
┌──────────────────────────────────┐
│ ■ 核心要点                        │  ← card-kicker (22px, 大写字母间距)
│ ───────────────────────────────── │
│  • 要点一                          │  ← card-body (30px)
│  • 要点二                          │
│  v = s / t                        │  ← formula (36px, 宋体, 主色)
└──────────────────────────────────┘
```

- 左侧 6px 主色边条
- 浅白背景，1.5px 边框
- 阴影：主色 9% 透明度，16px 模糊

### 小节 Chip (section-chip)

- 左侧：圆形编号（56px 直径，主色背景，白字）
- 右侧：圆角标签（白色背景，细边框）

### 标注 Callout

- 圆角矩形，主色边框
- 主色文字，白底半透明
- 2px 边框，10px 圆角

### 对比表 (compare-table)

- 表头两列分别用两种主色（概念 A 色 + 概念 B 色）
- 内容行交替浅背景
- 16px 圆角，整体阴影

## 六、安全区与布局

### 安全边距
- 所有重要内容距离边缘 ≥ 80px
- 底部字幕区域：底部 56px 起，高约 80px
- 顶部 chip 区域：顶部 48px 起，高约 60px

### 常见布局

| 布局 | 比例 | 适用场景 |
|------|------|---------|
| 左图右卡 | 左 55% + 右 45% | 概念讲解（最常用） |
| 左卡右图 | 左 45% + 右 55% | 对比第二个概念 |
| 居中标题 | 全宽居中 | 标题页、总结页 |
| 表格居中 | 表格宽度 1000-1200px | 对比表格页 |
| 上下结构 | 上 40% + 下 60% | 实验演示、时间轴 |
