# 场景积木库 · Scene Building Blocks

> 10 种常用教学场景模板，AI 可直接挑选组合。
> 每个积木包含：结构说明 + HTML/CSS 骨架 + 动效建议 + 适用知识点。

---

## 积木 01：标题引入页 (Title Scene)

**适用场景**：视频开头，引入主题

**结构**：
- 上方：主题图标 / emoji 组合（2-3 个视觉元素）
- 中间：大标题（主色，宋体粗体）
- 中下方：子概念胶囊标签（2-4 个）
- 底部：副标题 / 引言（小号，浅色）

**动效编排**：
1. 图标从上方弹跳入场（back.out, scale 0.7→1）
2. 主标题从下方滑入（power3.out, y:60→0）
3. 子概念胶囊 stagger 从两侧滑入
4. 副标题淡入

**HTML 骨架**：
```html
<div class="scene">
  <div class="scene-inner" style="flex-direction:column; justify-content:center; align-items:center; gap:40px;">
    <div id="s1-icon">🏃 ⚡ 🚗</div>
    <div class="t-hero">主题标题</div>
    <div id="s1-tags" style="display:flex; gap:18px;">
      <span class="section-chip" style="position:static;"><span class="chip-label">子概念1</span></span>
      <span class="section-chip" style="position:static;"><span class="chip-label">子概念2</span></span>
    </div>
    <div class="t-subtitle">—— 副标题 ——</div>
  </div>
</div>
```

## 积木 02：概念讲解页 (Concept Scene)

**适用场景**：讲解一个核心概念，是教学视频最常用的场景

**结构**：
- 左/右上：section-chip（编号 + 小节名）
- 左侧（60%）：SVG 示意图 + callout 标注
- 右侧（40%）：info-card（核心要点 + 公式）
- 底部：ghost 底纹大数字

**动效编排**：
1. 示意图像素描边绘制（stroke-dashoffset, power2.inOut）
2. 关键部件依次入场（stagger 0.1s）
3. callout 标注弹出（back.out, scale 0.5→1）
4. 卡片从右侧滑入（power3.out, x:80→0）
5. 公式行强调淡入

## 积木 03：对比表格页 (Compare Table Scene)

**适用场景**：对比两个易混淆概念，如速度 vs 加速度、电压 vs 电流

**结构**：
- 上方：对比标题（A vs B，两种主色分别标注）
- 中央：对比表格（2-4 行，维度 × 两个概念）
- 底部：记忆口诀 / 核心区别提示框

**表格行类型**：
- 定义行：一句话定义
- 公式行：公式对比（用不同颜色）
- 单位行：单位符号
- 误解行：常见错误认知 + 纠正

**动效编排**：
1. 标题从上方滑入
2. 表格整体淡入 + 微缩放
3. 表格行逐行 stagger 出现（每行 0.3s 间隔）
4. 底部提示框弹跳出现

## 积木 04：公式推导页 (Formula Derivation Scene)

**适用场景**：数学 / 物理公式推导过程

**结构**：
- 左侧：推导步骤编号（1→2→3→...）
- 中间：公式逐步展开
- 右侧：每步的文字说明卡片
- 底部：最终公式大字号强调

**动效编排**：
1. 起始公式淡入
2. 每步变换：旧公式淡出上移，新公式从下方淡入
3. 变换部分高亮闪烁（颜色脉冲 2 次）
4. 右侧说明卡片同步滑入
5. 最终公式放大强调

## 积木 05：实验演示页 (Experiment Scene)

**适用场景**：化学 / 物理 / 生物实验过程展示

**结构**：
- 左侧：实验装置 SVG 图（烧杯 / 弹簧 / 电路...）
- 右上：实验步骤文字
- 右下：实验现象观察卡片
- 底部：实验结论

**动效编排**：
1. 实验装置搭建（从下往上组装）
2. 实验过程动画（液体变色 / 弹簧拉伸 / 灯亮起）
3. 现象标注依次弹出
4. 结论卡片滑入

---

## 积木 06：生活应用页 (Real-world Scene)

**适用场景**：知识点在生活中的应用实例

**结构**：
- 上半部分：生活场景大图 / 图标组合（3-4 个应用场景图标）
- 下半部分：3 个应用卡片横排
  - 每个卡片：图标 + 场景名 + 一句话原理

**动效编排**：
1. 场景图标逐个弹入（stagger 0.2s, back.out）
2. 下方卡片从下往上 stagger 滑入
3. 卡片内文字依次出现

---

## 积木 07：问答互动页 (Q&A Scene)

**适用场景**：常见问题解答、易错点辨析

**结构**：
- 上方：问题气泡（大号，问号图标）
- 中间：思考 / 错误答案（划掉效果）
- 下方：正确答案卡片（绿色强调）
- 右下角：解释文字

**动效编排**：
1. 问题气泡弹出（弹性缩放）
2. 错误答案出现后划掉（line-through 动画）
3. 正确答案从下方滑入，伴随绿色对勾
4. 解释文字淡入

---

## 积木 08：时间轴页 (Timeline Scene)

**适用场景**：历史事件、发展过程、分阶段讲解

**结构**：
- 横向时间轴线（从左到右）
- 3-5 个时间节点（圆形节点 + 竖线 + 卡片）
- 节点上下交替排列，避免拥挤

**动效编排**：
1. 时间轴从左往右绘制（scaleX 0→1）
2. 节点逐个弹出（stagger 0.3s）
3. 每个节点的说明卡片从节点位置展开

---

## 积木 09：分类归纳页 (Classification Scene)

**适用场景**：知识体系梳理、分类总结

**结构**：
- 顶部：中心概念（大圆 / 大标题）
- 下方：2-3 个分支（每个分支一个分类卡片）
- 每个分支：分类名 + 2-3 个要点

**动效编排**：
1. 中心概念放大出现
2. 连接线从中心向各分支绘制
3. 分支卡片 stagger 弹入
4. 卡片内要点逐行出现

---

## 积木 10：总结收束页 (Summary Scene)

**适用场景**：视频结尾，回顾要点

**结构**：
- 上方：大数字 / 核心公式（2-3 个，左右排列）
- 中间：总结信息卡片（3-4 个要点）
- 底部：徽章行（4-6 个知识点徽章，icon + 文字）

**动效编排**：
1. 大数字从两侧飞入（y:-60→0, back.out）
2. 总结卡片从下方滑入
3. 徽章逐个弹跳入场（stagger 0.15s）
4. 最后整体轻微呼吸动效

## 场景搭配原则

| 视频类型 | 推荐场景组合（5 段式） |
|---------|----------------------|
| 概念辨析 | 标题 → 概念 A → 概念 B → 对比表格 → 总结 |
| 公式推导 | 标题 → 问题引入 → 推导过程 → 公式应用 → 总结 |
| 实验讲解 | 标题 → 实验装置 → 实验过程 → 现象分析 → 总结 |
| 生活科普 | 标题 → 概念讲解 → 生活应用 → 常见问答 → 总结 |
| 知识梳理 | 标题 → 分类 1 → 分类 2 → 时间轴/归纳 → 总结 |

**通用规则**：
- 标题段 ≈ 10-12 秒（钩子 + 主题介绍）
- 中间概念段每段 ≈ 12-15 秒（讲解清楚一个点）
- 对比/问答段 ≈ 12-15 秒（信息密度高，给足阅读时间）
- 总结段 ≈ 8-10 秒（快速回顾，留余韵）
