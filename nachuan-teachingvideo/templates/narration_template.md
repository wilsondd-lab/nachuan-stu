# 旁白文案模板 · Narration Template

> 7 段式教学视频旁白文案模板，AI 按此格式填写即可。
> 每段 30-50 字，口语化，像学长/学姐在讲解。

---

## 通用规范

### 字数与节奏
- 每段旁白 **30-50 字**（中文口语约 4-5 字/秒，对应 6-12 秒）
- 全片总时长 **60-90 秒**（7 段 × 平均 10 秒 ≈ 70 秒）
- 语速：约 **4.5 字/秒**（自然讲解语速，不要太快）

### 语言风格
- ✅ 口语化，像老师在讲，不是念课本
- ✅ 用"我们""你"拉近距离
- ✅ 短句为主，一句话讲一个意思
- ✅ 用连接词自然过渡："但是""所以""其实""换句话说"
- ❌ 不读公式（画面有公式，旁白用口语解释）
- ❌ 不说"接下来我们看""下面我们学习"
- ❌ 不用书面语和学术腔

### 数字读法
数字必须写成**读法**，不要写符号：
- `3×10⁸` → "三亿米每秒"
- `9.8 m/s²` → "九点八米每二次方秒"
- `v = s/t` → "速度等于位移除以时间"

## 7 段文案模板（误解优先型）

> 最常用的结构：用常见误解开头，抓住注意力，再讲正确概念。

| 段落 | 角色 | 字数 | 写作要点 | 示例模板 |
|------|------|------|---------|---------|
| **S1 标题** | 抛误解 + 点题 | 35-45 字 | 第一句就制造认知冲突，点明主题 | "很多人以为速度快就是加速度大，其实这两个概念完全不一样。今天我们把它们彻底分清。" |
| **S2 概念1** | 讲第一个概念 | 40-50 字 | 定义 + 公式的口语解释 + 单位 | "先说速度。速度描述物体运动的快慢，等于位移除以时间，单位是米每秒。" |
| **S3 概念2** | 讲第二个概念 | 40-50 字 | 同上，注意换个角度讲，不要重复句式 | "再看加速度。它描述的是速度变化的快慢，速度变化量除以时间，单位是米每二次方秒。" |
| **S4 对比/辨析** | 指出关键区别 | 40-50 字 | 直接对比，用生活例子说明 | "关键区别是：速度是走多快，加速度是变得多快。匀速飞行的飞机速度很大，但加速度是零。" |
| **S5 深化/应用** | 进阶理解 | 35-45 字 | 讲一个容易错的点或生活应用 | "刚启动的赛车，加速度很大但速度还很小，因为速度还没来得及积累。" |
| **S6 总结/口诀** | 方法总结 | 30-40 字 | 给一个记忆方法或判断技巧 | "记住一句话：同向加速，反向减速。看方向就知道速度在变大还是变小。" |
| **S7 收尾** | 收束 + 余韵 | 25-35 字 | 回顾核心，留下记忆点 | "速度和加速度，一个是快慢，一个是变化快慢，下次别再搞混啦。" |

## 变体模板：引导发现型

> 适合需要推理过程的概念，用问题引导学生思考。

| 段落 | 角色 | 示例 |
|------|------|------|
| S1 | 提问题 | "为什么冬天哈气会有白雾？这背后藏着一个重要的物态变化知识。" |
| S2 | 朴素尝试 | "你可能觉得是水蒸气，但水蒸气其实是看不见的，那白雾到底是什么？" |
| S3 | 遇到困难 | "要搞清楚这个问题，我们得先明白：气体、液体、固体之间是怎么转换的。" |
| S4 | 关键洞察 | "答案是液化。高温的水蒸气遇到冷空气，瞬间变成了小水滴——这就是白雾。" |
| S5 | 逐步构建 | "物质从气态变成液态叫液化，液化需要放热。所以哈气时手会觉得暖。" |
| S6 | 推广应用 | "生活中液化现象随处可见：冰箱里拿出的饮料冒汗、早晨的露水，都是同一个道理。" |
| S7 | 一般化总结 | "记住：气体遇冷液化放热。下次看到白雾，你就知道是怎么回事了。" |

## 变体模板：递进讲解型

> 适合系统性知识，层层深入。

| 段落 | 角色 | 示例 |
|------|------|------|
| S1 | 总览 | "今天我们系统学习声音。声音是怎么产生的？怎么传播的？又有哪些特性？" |
| S2 | 基础概念 | "声音由物体振动产生。振动停止，发声也停止。这个振动的物体叫声源。" |
| S3 | 进阶1 | "声音的传播需要介质。固体、液体、气体都能传声，但真空不能传声。" |
| S4 | 进阶2 | "声音在不同介质中速度不同。固体最快，液体次之，气体最慢，空气中约每秒三百四十米。" |
| S5 | 核心特性 | "声音有三个特性：音调、响度、音色。音调由频率决定，响度由振幅决定。" |
| S6 | 综合应用 | "利用这些特性，我们可以解释很多现象：为什么男生声音低沉女生尖细，为什么听诊器能听心跳。" |
| S7 | 总结 | "从产生到传播再到特性，声音的世界就讲完了。掌握这三点，声学基础就扎实了。" |

## 画面描述模板（visual 字段）

每段旁白配一段画面描述，格式如下：

```
【布局】左图右卡 / 居中 / 上下结构
【主体】画面核心是什么（示意图/卡片/数字/表格）
【出场顺序】
  1. XX 从左绘制入场（0.2s）
  2. XX 弹出（0.6s）
  3. 卡片从右侧滑入（0.8s）
【关键动效】绘制 / 滑入 / 弹出 / 缩放 / 闪烁
【callout标注】位置 + 文字
【配色】主色 + 对比色
```

**示例**：
```
【布局】左图右卡
【主体】左侧小车在道路上运动的SVG示意图，右侧速度概念卡片
【出场顺序】
  1. 道路从左到右绘制（0.2s）
  2. 小车从左侧驶入（0.4s）
  3. 速度箭头伸长 + v标注弹出（0.9s）
  4. 信息卡片从右侧滑入（0.6s）
【关键动效】stroke-dashoffset 绘制、x 方向滑入、scale 弹出
【callout标注】小车右上方："匀速运动"
【配色】暖橙主色
```

## 质量自检清单

写完文案后，逐项检查：

- [ ] 每段 30-50 字（过长要精简，过短要补充）
- [ ] 口语化，没有书面语和学术腔
- [ ] 公式用口语解释，不直接念符号
- [ ] 数字写成读法
- [ ] 有连接词过渡，不是"然后然后然后"
- [ ] 钩子段（S1）制造了信息缺口或认知冲突
- [ ] 结尾段（S7）回顾了核心，有记忆点
- [ ] 全片新概念不超过 5 个
- [ ] 画面描述和旁白讲的是同一个东西（时空同步）
